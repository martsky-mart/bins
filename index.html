    Pick a blue or brown bin day from the council calendar — not a black-bin day. That fixes the whole cycle.
    Collections run weekly: black general waste one week, a recycling bin the next. The recycling colour alternates blue then brown.

    Garden waste is shown on black-bin weeks — check your calendar and let me know if it differs.

    Notifications ▾

    Get a phone notification the evening before collection, even when this app is closed. Setup is two steps: enable it here, then paste one file into GitHub once.

    Finish setup on GitHub (one-off)

    1. Tap Copy below.
    2. Tap Open subscription.json on GitHub — it opens the file already in edit mode.
    3. Select all the existing text in the box and paste over it.
    4. Scroll down and tap Commit changes…, then confirm.
